# 📊 Prop Trading Firm Dashboard App

A modern, mobile-first **Prop Trading Dashboard Application** designed to provide traders with a clean and intuitive interface to track their account performance, evaluation progress, and trading rules.

## 🚀 Features

- 💰 Account Balance & Evaluation Tracking
- 📈 Real-time Progress Bars (Profit Target, Drawdown, etc.)
- 🔐 Authentication UI (Login / Signup)
- 📊 Trading Metrics Visualization
- ⚡ Smooth UI Animations & Micro-interactions
- 📱 Fully Responsive (Android + iOS Friendly Layout)
- 🌙 Modern Dark Fintech Theme

## 🎯 Purpose

This app simulates a **proprietary trading platform dashboard**, where traders are evaluated based on predefined rules before receiving funded accounts.
